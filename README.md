# simplereader
#### "简阅" 网络小说阅读器仿照["任阅" 网络小说阅读器](https://github.com/JustWayward/BookReader)，实现追书推荐收藏、书籍/标签检索、读书等功能!

## 介绍
项目基于ReactNative + Redux的框架开发。项目中的API均来自追书神器，纯属共享学习之用，不得用于商业！！

android下载地址: http://oouqiilcr.bkt.clouddn.com/book_1.0_2017-04-28.apk

## 安装

1. git clone https://github.com/jsntjinjin/simplereader.git
2. cd simplereader
3. npm install
4. react-native link
5. react-native run-ios / react-native run-android
